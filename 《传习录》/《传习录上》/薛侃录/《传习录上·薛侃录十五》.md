# 原文
问：“上智下愚如何不可移[^1]？”

先生曰：“不是不可移，只是不肯移。”
# 笔记

# 题解
本条明圣、愚之异同。在阳明心学，圣、愚之同在于良知自足，每人都有成圣的内在潜质。圣、愚之异在于圣人能够致良知，愚人不能致良知。故从良知普遍性而言，本无圣、愚之别，从功夫而言，在于是否肯真切去做功夫。
# 译文
薛侃问：“孔子为什么说，最聪明的人和最愚笨的人，他们的性情都不能改变呢？”

先生说：“并不是不能改变，而是不愿改变。”
# 注释

[^1]: 上智下愚如何不可移：语自《论语·阳货》：“子曰：‘唯上智与下愚不移。’”
